## 📌 매매 전략

### 매수 조건
- 거래대금 상위 50위 종목
- 당일 등락률 ≥ +5%
- 당일 거래량 ≥ 전일 거래량 × 2
- 상한가 종목 제외 (등락률 ≥ 29%)
- 변동성 필터: 35봉 이내 고가 +9.5% 이상 3일 이상 종목 제외
- 테마 당일 등락률 ≥ +3%
- 매수금액: 전체 자산의 2%

### 매도 조건
- 익절: +10% 즉시 매도
- 손절: -5% 즉시 매도
- 시간매도: 다음 거래일 09:10 매도

### 거래 비용
- 수수료: 0.015%
- 매도세: 0.18%
- 슬리피지: 0.1%

## 🛠 환경
- Python 3.11.9 32-bit (키움 API)
